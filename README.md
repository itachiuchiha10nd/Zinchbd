🎂 ZNICHBD (Birthday Surprise Project)

A fully interactive birthday surprise website with animations, gifts, flipbook, videos, and a final message.

💖 Features

- 🔐 Password entry system
- ✉️ Envelope opening animation
- 💌 Letter with typing + auto-fit text
- 🎂 Cake with candle + blow effect
- 🎈 Balloons, fireworks, and sound effects
- 🎁 4 Gift boxes:
  - Gift 1 → Flipbook
  - Gift 2 → Video (auto-play)
  - Gift 3 → Custom page
  - Gift 4 → Custom page
- 🎯 Final message unlock after all gifts
- 💕 Beautiful UI with animations

🚀 How to Run

1. Download or clone the project
2. Keep all files in the correct structure
3. Open "index.html" in browser
4. Enter password to start experience

⚙️ Important Notes

- File names must match exactly (case-sensitive)
- GIF/video files should not be corrupted
- Works best on Chrome (mobile + desktop)
- Autoplay videos may require user interaction (browser policy)

🎁 Customization

You can easily modify:

- Text messages (inside JS "msg")
- Images (assets folder)
- Videos and flipbook content
- Colors and styles (CSS)
